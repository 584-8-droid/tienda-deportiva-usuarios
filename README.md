# Prototipo - Tienda Deportiva 🏆

Este es el prototipo inicial del sistema de **Gestión de Usuarios** para la aplicación de la tienda deportiva SportZone. 

El sistema demuestra la funcionalidad completa de un CRUD (Crear, Leer, Eliminar) de usuarios, junto con un inicio de sesión funcional, usando una base de datos local.

## Tecnologías Utilizadas 🛠️
- **Backend:** Python con Flask
- **Base de Datos:** SQLite (SQLAlchemy)
- **Frontend:** HTML, CSS (Diseño personalizado) y Vanilla JavaScript

## Cómo ejecutar el proyecto 🚀
1. Instalar las dependencias: `pip install -r requirements.txt`
2. Ejecutar el servidor: `python main.py`
3. Abrir en el navegador: `http://127.0.0.1:8000`
4. Usuario administrador por defecto: `admin` / `admin123`
